# Contributor Code of Conduct

We expect all participants to respect and support the values of our community: inclusivity, respect, and collaboration. By contributing to this project you agree to follow this code of conduct.

Our standards
- Be respectful and courteous to others.
- No harassment, hate speech, or abusive behavior.
- Be constructive in feedback; focus on ideas, not individuals.
- Respect privacy and personal information.
- Report any issues or violations through the appropriate channels (issues or PRs).

Enforcement
- This code of conduct is enforced by project maintainers. Violations may result in removal from the contributor/discussion channels or repository access.

Scope
- This code applies to all project communications, including issues, PRs, commit messages, and any discussions related to the project.

Acknowledgement
- This Code of Conduct is adapted from the Contributor Covenant 1.4.
